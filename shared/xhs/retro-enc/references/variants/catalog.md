# 器物图录（Catalog）
工具百科、器具展示、收藏鉴赏。

## 核心视觉
- 多件器物整齐排列如博物馆展柜
- 精细描绘+光影质感
- 标注尺寸/材质/年代
- 分类编号系统
- 收藏鉴赏仪式感

## Elements
```yaml
layout: center-radial | panoramic
annotations: catalog-labels, museum-specimen-cards
data_viz: [size-scale, material-legend]
decorations: [catalog-numbers, quality-badges, museum-border]
```

## Prompt 片段
```
A vintage catalog illustration displaying [N] [subject] specimens on deep 
aged parchment, museum collection plate style. Each item meticulously painted 
with material texture. Catalog-style numbering and labels. Size scale.
```

## 示例
「中国传统茶具图录」「老木匠的工具箱」「90后的童年零食」
