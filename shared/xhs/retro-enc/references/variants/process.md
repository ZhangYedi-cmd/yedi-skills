# 流程图解（Process）
制作步骤、行为过程、时间线。

## 核心视觉
- 编号步骤①②③ + 虚线箭头动线
- 中心主体动态姿态
- 触发条件列表/时间标记
- 场景小图各阶段
- 幽默金句

## Elements
```yaml
layout: sequential | timeline-flow
annotations: numbered-steps, trajectory-paths
data_viz: [timeline, trigger-conditions]
decorations: [numbered-circles, arrows, stage-vignettes]
```

## Prompt 片段
```
A vintage scientific illustration analyzing [process] on weathered parchment. 
Central [subject] in dynamic pose with numbered dotted-line trajectory paths. 
Scene vignettes showing triggers and stages. Clock/timeline. Humorous caption.
```

## 示例
「猫咪疯跑行为图鉴」「手冲咖啡的七个步骤」「包饺子的科学」
