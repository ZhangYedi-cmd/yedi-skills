# 融合对比（Fusion）
混血分析、跨界对比、A+B=C。

## 核心视觉
- 顶部：A + B 剪影对比
- 中间：融合体 C 大图 + 特征标注
- 两侧：行为对比场景
- 底部：数据对比图表
- 视觉公式：A + B = C

## Elements
```yaml
layout: fusion-plate | side-by-side
annotations: trait-inheritance-lines
data_viz: [comparison-table, radar-chart, size-graph]
decorations: [visual-equation, size-silhouettes]
```

## Prompt 片段
```
A vintage comparison plate illustrating fusion of [A] and [B] into [C] on 
parchment. Top: silhouette comparison. Center: large hybrid [C] with trait 
annotations. Sides: behavioral vignettes. Bottom: personality charts.
```

## 示例
「缅因猫＋橘猫＝缅橘」「中餐＋西餐＝Fusion菜」
