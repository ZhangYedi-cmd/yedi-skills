# 搭配指南（Pairing）
食物搭配、工具组合、穿搭推荐。

## 核心视觉
- 中心主体（碗/盘/物）+ 周围8-12搭配项
- 每项独立精绘+名称标签
- 可分区域标记
- 红印章点缀、淡传统山水背景

## Elements
```yaml
layout: grid-surround
annotations: chinese-style name-tags
decorations: [red-seal-stamps, category-ribbons, faint-landscape-bg]
```

## Prompt 片段
```
A vintage Chinese encyclopedia illustration showcasing [subject] and pairings 
on aged paper. Central [subject] in ornate container, surrounded by [N] 
companion items with Chinese name labels. Red seal stamps. Faint traditional 
landscape background.
```

## 示例
「白粥的绝配有哪些」「火锅蘸料的灵魂配方」「文具控的桌面必备」
