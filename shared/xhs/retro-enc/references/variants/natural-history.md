# 自然史博物（Natural History）
品种展示、分类图鉴、家族谱系。

## 核心视觉
- 多标本排列，中心最大，周围缩小
- 每标本独立完整+名称标签
- 分类标签系统

## Elements
```yaml
layout: center-radial | panoramic
annotations: classification-labels, hand-written name tags
decorations: [red-seal-stamps, victorian-border, corner-flourishes]
```

## Prompt 片段
```
A vintage natural history plate depicting [N] distinct varieties of [subject] 
on aged parchment, 19th-century museum specimen plate style. Each specimen in 
watercolor with Chinese name label. Central specimen largest and most detailed.
```

## 示例
「橘猫的基因有多强大」「中国茶叶图鉴」「多肉植物入坑指南」
