# 解剖标注（Anatomy）
行为分析、结构拆解、特征解读。

## 核心视觉
- 大主体居中 + 6-12条标注线
- 局部放大圆(1-3个)
- 行为场景小图
- 底部数据面板 + 幽默金句框

## Elements
```yaml
layout: annotated | dashboard
annotations: anatomical-lines (6-12), circular-magnification (1-3)
data_viz: [star-ratings, radar-chart, bar-chart]
decorations: [magnified-insets, behavioral-vignettes, quote-box]
```

## Prompt 片段
```
A detailed vintage scientific illustration analyzing [subject] on weathered 
parchment. Large central figure with anatomical-style annotation lines to 
Chinese labels. Magnified circular detail insets. Behavioral vignettes. 
Rating charts in vintage style.
```

## 示例
「缅因猫＋橘猫＝缅橘」特征拆解、「猫咪疯跑行为图鉴」、「程序员的一天」
