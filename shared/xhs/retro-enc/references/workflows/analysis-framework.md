# Analysis Framework 主题分析框架

## 分析维度

### 1. 核心对象
主体是什么？属什么大类？具象还是抽象？

### 2. 信息类型 → Variant
| 信号词 | Variant |
|--------|---------|
| 品种/种类/分类/大全 | natural-history |
| 结构/原理/分析/拆解 | anatomy |
| 搭配/配什么/绝配 | pairing |
| 混合/融合/对比/VS | fusion |
| 步骤/流程/过程 | process |
| 工具/器具/收藏 | catalog |

### 3. 内容深度 → 页数
1-2维 → 3张, 3-4维 → 4-5张, 5+维 → 6-8张

### 4. 调性
严肃科普 / 幽默反差 / 温馨怀旧 / 实用指南

### 5. 幽默潜力
| 反差类型 | 示例 |
|----------|------|
| 外表vs内在 | "外表北境领主，内心零食收割机" |
| 学术vs日常 | *Freneticus Domesticus* 描述疯跑 |
| 数据vs感性 | 严肃星级量化"撒娇指数" |

## 输出格式
```markdown
# 主题分析：[主题]
## 核心对象 / 信息类型 / 推荐方案（变体+构图+页数）
## 幽默切入点 / 伪学名建议(×3) / 标题建议(×3)
```
