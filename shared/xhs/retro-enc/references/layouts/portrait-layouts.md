# Portrait Layouts 竖版 (9:16 / 3:4)

## center-radial 中心辐射
```
Central large [subject] on aged parchment, surrounded by [N] smaller labeled 
specimens in semi-circular layout. Each with hand-written Chinese name label. 
Bold Chinese title「[标题]」at top. Faux-Latin at corners.
```
主体35-45%, 环绕6-10个小图

## annotated 解剖标注
```
Large central [subject] on weathered parchment with [N] anatomical annotation 
lines to Chinese labels. [M] magnified circular detail insets. Behavioral 
vignettes in corners. Data panel at bottom: star ratings and charts.
Bold Chinese title. Quote box with witty caption.
```
主体40-50%, 标注6-12条, 放大圆1-3个

## grid-surround 九宫环绕
```
Central [subject] in ornate container on aged parchment, surrounded by [N] 
companion items in symmetrical grid. Each with Chinese name label. Red seal 
stamps as accents. Faint traditional Chinese landscape background.
```
主体30-40%, 配件8-12个, 印章2-3个

## fusion-plate 融合图版
```
Top: silhouette comparison of [A] and [B] with plus/equals signs leading to [C].
Center (40%): large hybrid [C] with trait annotation lines in Chinese.
Sides: behavioral vignettes. Bottom: comparison chart, star ratings, radar diagram.
```

## sequential 动态序列
```
Central [subject] in dynamic pose with numbered (①②③) dotted-line trajectory 
paths. Scene vignettes at corners for triggers/aftermath. Anatomical cross-section 
detail. Clock element. Humorous caption box at bottom.
```
轨迹节点5-9个, 场景3-5个

## 通用尾部（附加到所有竖版 prompt）
```
Hand-painted watercolor and colored pencil, 19th-century natural history style.
Warm sepia, amber, cream on yellowed antique paper. Worn edges.
```
