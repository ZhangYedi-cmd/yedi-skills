# Landscape Layouts 横版 (16:9 / 4:3)

## side-by-side 左右分栏
```
Landscape (16:9). Left: detailed [A] with Chinese annotations. Right: [B] with 
matching annotations. Center: ornamental divider/"VS". Bottom: comparison table 
with bar charts and star ratings. Title banner at top.
```

## panoramic 全景展开
```
Landscape (16:9). Wide panoramic lineup of [N] [subject] varieties side by side 
on parchment. Central specimen largest, flanking progressively smaller. Each with 
Chinese name label below. Botanical border top and bottom.
```

## timeline-flow 时间流程
```
Landscape (16:9). Step-by-step [process] left to right. [N] stages connected by 
decorative arrows. Each stage: watercolor vignette + circled number + Chinese label.
Timeline bar at bottom. Materials panel bottom-left.
```

## scene-map 场景地图
```
Landscape (16:9). Vintage map of [scene] as hand-painted bird's-eye view. [N] 
labeled points with numbered flags + illustration icons. Detail vignettes in 
margins. Compass rose. Cartographic border with scale.
```

## dashboard 仪表盘
```
Landscape (16:9). Left 40%: large detailed [subject]. Right 60%: vintage data 
visualizations — bar charts, radar diagrams, star ratings, ranked lists. 
Hand-drawn infographic style with sepia ink. Decorative dividing rules.
```

## 通用尾部（附加到所有横版 prompt）
```
Hand-painted watercolor and colored pencil, 19th-century natural history style.
Warm sepia, amber, cream on yellowed antique paper. Worn edges.
```
