# Color Palettes 色彩体系

## 核心色板（共享）
| 角色 | 色值 | 用途 |
|------|------|------|
| 纸张 | #F5E6D3 羊皮纸, #FFFFF0 象牙, #E8DCC8 淡褐 | 背景 |
| 主色 | #8B6914 暖棕, #D2691E 焦糖, #FFBF00 琥珀 | 主体 |
| 点缀 | #C55A5A 朱红, #2C3E6B 深蓝, #2E4A3E 墨绿, #B8860B 旧金 | 印章/学名/边框 |
| 禁用 | 霓虹, 荧光, 纯黑大面积, 冷白 | — |

## Prompt 色彩模板
```
Color palette: warm sepia and cream on aged parchment. Primary: warm browns,
caramel, amber. Background: yellowed antique paper (#F5E6D3). Accents:
vermillion red for stamps, deep blue for Latin text.
AVOID: neon, fluorescent, pure black fills, cold white.
```

## 变体色调偏向
- natural-history: 暖棕 + 橄榄绿
- anatomy: 焦糖 + 朱红标注
- pairing: 暖色 + 食物本色
- fusion: A/B各自色系 + C融合色
- process: 琥珀 + 朱红编号
- catalog: 旧金 + 深棕
